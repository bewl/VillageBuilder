# ?? Tile Inspection - Quick Start Guide

## How to Use

### **Basic Controls**

**Click on any tile:**
- **People present?** ? Selects the person/family
- **Building present?** ? Selects the building  
- **Empty tile?** ? Shows tile inspection!

**Hold Ctrl + Click:**
- **Forces tile inspection** (even with people/buildings)
- Use this when you want to inspect the tile itself

**Hover over tile:**
- **Quick tooltip** appears
- Shows terrain, decorations, people count
- No click needed!

## Selection Priority

The game follows this order:

```
1. Ctrl held? ? Always select tile
2. People present? ? Select person
3. Building present? ? Select building  
4. Empty tile? ? Select tile
```

**Why this order?**
- People need frequent attention
- Buildings are important structures
- Tiles are the foundation (always accessible with Ctrl!)

## Troubleshooting

**Q: I clicked a tile but nothing happened**
- A: You probably clicked sidebar - try clicking map area

**Q: I want to inspect a tile but people are on it**
- A: Hold Ctrl while clicking!

**Q: Tooltip doesn't show**
- A: Make sure you're hovering over the map, not sidebar

**Q: Can't see all decorations**
- A: If >8, inspector shows "... and X more"

**Q: How do I get back to person selection?**
- A: Just click the person name in tile inspector!
